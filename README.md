## Description

Template DDD TypeScript + Cucumber + RabbitMQ + NestJs + PrismaORM + PostgresSQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests gherkins cucumber
$ npm run test:ecommerce:backend:rest:features

# test coverage
$ npm run test:cov
```
